Visualization Overview

For this project, we created the family tree of the Trump family.

The given data set was changed by creating the new data structure:

  • Each node has 4 attributes:
    • name: name of a person or a partnership, i.e., two people's names separated by a dash
    • gen: number for a generation (3 identifies a root node, 2 - a child, and so on). Numbers go in a decreasing order
    • blood: true identifies that a person is related by a blood line to the Trump's family
    • type: person | partnership
  • Each link has 3 attributes:
    • source: where a line comes from
    • target: where a line goes to
    • hidden: true identifies that a link is not visible

Discussion

1. Motivate your choices (e.g. choice of color/marks, overall layout etc).

Since we show a representation of the Trump's family tree, we decided to use empty circles for people who are not related to Trump's family by blood line. A small empty black circle with grey lines to people identifies a family.

Different color for circles were chosen according to a generation. We decided to use a saturation of red color, i.e., lighter color stands for younger generation. In this way, it is easier to find, for example, a child or grandchild, because chroma is perceived as ordered list unlike hue. Also by using different colors for each generation we would end up with too colorful family tree with no connection between colors.

In addition to this, some people are color blind so different colors are not helpful for them while separating generations. We tested our red color choice against a color-blind simulator in www.rehue.net. As it can be seen from Figure 1 and Figure 2, colorful circles cause problems as two different colors, e.g., second image in Figure 2, are understood as the same color.

However color saturation is not the best option either, if we have a lot of generations. In this case, it is worth considering having different type of tree like the one in http://bl.ocks.org/robschmuecker/7926762 where you can collapse or expand nodes.

To make the separation between generations even more obvious we chose different sizes of circle as well.
